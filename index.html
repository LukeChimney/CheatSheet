GIT-Befehle

Befehl Beschreibung
git init GIT inizialisieren

Terminal-Befehle

Befehl Beschreibung
pwd Zeigt das aktuelle Arbeitsverzeichnis an/td>
ls Auflistung aller Verzeichnisse
ls -a Wie "ls", jedoch mit versteckten Datein
ls -l Wie "ls -a", jedoch mit Zusatzinformationen
touch Datei erstellen
mkdir Ordner erstellen
mkdir -p xx/yy/zz Ordner mit Unterordnern erstellen
cd Verzeichnis wechseln
mv quelle ziel Datei verschieben
cp quelle ziel Datei kopieren
rm Datei löschen
rmdir Verzeichnis löschen
